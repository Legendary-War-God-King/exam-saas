# Agent 检查点

Huang-bao-sheng Phase 4/6/7 可选的 Agent 辅助：

| Phase | Agent | 何时用 |
|---|---|---|
| Phase 4 (实现) | Minimal Change Engineer | 改动复杂或不放心时 |
| Phase 6 (审查) | Code Reviewer | 代码审查第二意见 |
| Phase 7 (验证) | Reality Checker + webapp-testing | 前者证据审计，后者浏览器验证 |
| Sprint 里程碑 | SRE | 性能/SLO/灾备检查 |

用户说不用就跳过，不是强制。
