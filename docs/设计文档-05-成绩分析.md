# 成绩分析模块 设计文档

> V1.0 | 2026-05-28

## API

```
GET /exams/:id/results        考生成绩列表 (分页)
GET /exams/:id/statistics     考试统计 (平均分/最高/最低/及格率/分布)
GET /exams/:id/analysis       题目分析 (正确率/选项分布/难度系数)
GET /exams/:id/export         导出 Excel
```

## 统计维度

- 应考/实考/缺考人数
- 平均分 / 最高分 / 最低分 / 中位数
- 及格率 (score >= passScore)
- 分数段分布: 0-59 / 60-69 / 70-79 / 80-89 / 90-100

## 题目分析

- 每道题的正确率
- 单选/多选: 选项分布 (A:30%, B:25%, C:45%)
- 判断题: 正确率
- 难度系数 = 1 - 正确率

## Excel 导出

- 列: 学号, 姓名, 班级, 分数, 用时, 是否及格
- 文件名: exam-{id}-{date}.xlsx
