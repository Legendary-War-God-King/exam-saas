# 题库管理模块 设计文档

> 版本: V1.0 | 日期: 2026-05-28 | 状态: 设计完毕

## 1. 数据模型

### QuestionBank

| 字段 | 类型 | 说明 |
|------|------|------|
| id | uuid (PK) | |
| tenant_id | uuid (FK → Tenant) | 租户隔离 |
| name | String | 题库名称 |
| description | String? | 描述 |
| deleted_at | DateTime? | 软删除 |
| created_at | DateTime | |
| updated_at | DateTime | |

### Question

| 字段 | 类型 | 说明 |
|------|------|------|
| id | uuid (PK) | |
| bank_id | uuid (FK → QuestionBank) | 所属题库 |
| type | Enum(SINGLE_CHOICE/MULTI_CHOICE/TRUE_FALSE/FILL_BLANK) | 题型 |
| content | String | 题干 |
| options | Json? | 选项 (单选/多选时使用) |
| answer | String | 正确答案 |
| difficulty | Int? | 难度 1-5, 默认=1 |
| tags | String[]? | 标签 |
| deleted_at | DateTime? | 软删除 |
| created_at | DateTime | |
| updated_at | DateTime | |

## 2. API 设计

### 题库

```
POST   /api/v1/question-banks                 创建题库
GET    /api/v1/question-banks                 列表 (search)
GET    /api/v1/question-banks/:id             详情
PATCH  /api/v1/question-banks/:id             更新 (name/description)
DELETE /api/v1/question-banks/:id             软删除 (设 deletedAt)
```

### 题目

```
POST   /api/v1/question-banks/:bankId/questions         创建题目
GET    /api/v1/question-banks/:bankId/questions         列表 (?page=1&limit=20&type=&difficulty=&search=)
GET    /api/v1/questions/:id                             详情
PATCH  /api/v1/questions/:id                             更新
DELETE /api/v1/questions/:id                             软删除
POST   /api/v1/question-banks/:bankId/questions/import  Excel 批量导入
```

## 3. Excel 导入

- 接受 `.xlsx` 文件 (multer)
- 列: 题型 | 题干 | 选项A | 选项B | 选项C | 选项D | 答案 | 难度 | 标签
- 校验: 题型必填且合法、题干必填、答案必填
- 返回: `{ imported: 15, errors: [{row: 3, reason: "答案不能为空"}] }`
- 单次上限 200 行

## 4. 安全

- 所有查询加 tenant_id 隔离
- 软删除: 查询强制 `WHERE deleted_at IS NULL`
- 批量导入: 单租户单次 200 行上限
