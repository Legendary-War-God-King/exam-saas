# 学生端模块 设计文档

> V1.0 | 2026-05-28

## 数据模型

Student: id, tenant_id, name, student_no (学号/准考证号), class
ExamRecord: id, exam_id, student_id, start_time, end_time, score, status(IN_PROGRESS/SUBMITTED/CANCELLED)
Answer: record_id, question_id (复合PK), selected_answer, correct, time_spent

## 学生认证

- 学生无密码, 通过 学号(studetNo) + 考试码(examCode) 登录
- 考试码: 6位随机数字, Exam 发布时生成存到 exam.access_code
- POST /student/auth → 验证码+学号→签发 Student JWT (studentId, tenantId, examId, type:"student")
- Student JWT 有效期: 考试时长+30分钟缓冲

## 考试执行 API

```
POST /student/auth                      登录 (body: {studentNo, code})
GET  /student/exams/:examId/questions   获取题目 (不含答案)
POST /student/exams/:examId/answer      提交单题答案 (body: {questionId, selectedAnswer, timeSpent})
POST /student/exams/:examId/submit      交卷
GET  /student/exams/:examId/result      查看成绩
```

## 规则

- 提交答案: 自动创建或更新 ExamRecord (首次提交→IN_PROGRESS+记录startTime)
- 交卷: ExamRecord→SUBMITTED+记录endTime+自动算分
- 算分: 遍历Answer, 比对Question.answer→correct=true/false→score=Σ(correctQuestion.score)
- Student JWT: 只能调 /student/* 路由, Guard检查type==="student"
