# 轻量级在线考试 SaaS — 简历项目总结

> 独立全栈开发 | 34 commits | 2026 年 5 月

## 项目一句话

轻量级在线考试 SaaS 平台，支持培训机构/学校完成"出题→组卷→考试→改卷→分析"全流程闭环。

## 技术栈

- **后端:** NestJS 11 + TypeScript + Prisma 7 + PostgreSQL 17 + Redis 7
- **前端:** Next.js 14 (Pages Router) + React 18 + Tailwind CSS 3 + Zustand + Axios
- **基础设施:** Docker Compose (PostgreSQL + Redis + Nginx) + GitHub Actions CI/CD + Sentry 错误追踪
- **实时通信:** Socket.IO WebSocket (监考面板)

## 功能模块

| 模块 | 内容 |
|------|------|
| Multi-Tenant 认证 | JWT 双 Token (Access 15min + Refresh 7d)，BCrypt 密码哈希，RBAC (ADMIN/TEACHER) |
| 题库管理 | 单选/多选/判断/填空，Excel 批量导入，软删除，标签+难度分级 |
| 考试管理 | 手动组卷 + 智能随机抽题，发布生成 6 位考试码，防作弊开关 |
| H5 学生端 | 学号+考试码登录，倒计时答题，visibilitychange 切屏检测，交卷自动评分 |
| 成绩分析 | 平均分/最高分/最低分/中位数/及格率/分数段分布，题目正确率+选项分布，Excel(xlsx) 导出 |
| WebSocket 监考 | 学生心跳上报(10s TTL)，在线考生实时人数，作弊事件推送 |
| 安全设计 | OWASP API Top 10 应对：全局 TenantGuard 租户隔离、Helmet CSP、Redis 限流(5/min)、审计日志 |

## 技术亮点

1. **Multi-Tenant 数据隔离** — 所有查询强制追加 `WHERE tenant_id`，Guard 层自动注入租户上下文，杜绝跨租户数据泄露
2. **JWT 双 Token 轮换** — Access Token 15 分钟过期 + Refresh Token 7 天，payload 含 `type` 字段防止 token 混用
3. **防作弊系统** — 学生端 visibilitychange+blur+focus 监听，Redis 心跳 10s 上报，超 30s 无心跳标记异常
4. **自动评分** — 交卷时服务端逐题比对答案，计算总分，题目正确率实时更新
5. **Prisma 7 + 驱动适配器** — 使用 @prisma/adapter-pg 直连 PostgreSQL，避免 ORM 性能损耗
6. **ESLint strictTypeChecked** — 全量 TypeScript 严格模式，零 any 类型，DTO 全面 class-validator 校验
