# 考试管理模块 设计文档

> V1.0 | 2026-05-28

## API

```
POST   /api/v1/exams                    创建考试
GET    /api/v1/exams                    考试列表 (?status=)
GET    /api/v1/exams/:id                考试详情 (含题目列表)
PATCH  /api/v1/exams/:id                更新考试信息
DELETE /api/v1/exams/:id                软删除
PATCH  /api/v1/exams/:id/publish        发布 (DRAFT→PUBLISHED)
POST   /api/v1/exams/:id/questions      添加题目 (body: { questionId, score, sortOrder })
DELETE /api/v1/exams/:id/questions/:qid 移除题目
PATCH  /api/v1/exams/:id/questions/:qid 更新分值/排序
```

## 规则

- 创建: ADMIN/TEACHER, DRAFT 状态
- 发布: 必须有 ≥1 道题目, DRAFT→PUBLISHED
- 已发布不能修改题目列表, 需先取消发布
- 软删除: DRAFT 可删, PUBLISHED 不可删
- 返回题目列表时按 sortOrder 升序

## ExamStatus

```
DRAFT → PUBLISHED → IN_PROGRESS → FINISHED
```
